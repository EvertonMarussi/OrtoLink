### Criando diretórios - V0.1.0
- Estrutura Básica de diretórios do projeto vue.
- public: Arquivos publicos como o index.html e o icone da página
- node_modules: Todas as dependências do projeto
- src: Todos os arquivos do projeto
- assets: icones, imagens e fotos
- components: Componentes do frontend, como botões, formulários, cards, gráficos. Utilizado para modularizar o código
- config: Arquivos de configuração, como as configs da minha API, config de rotas
- layouts: A estrutura de layout das páginas ou plataforma
- utils: funções comuns que podem ser reutilizadas ao longo do projeto
- views: Páginas do frontend.
